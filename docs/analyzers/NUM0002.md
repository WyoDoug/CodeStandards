# NUM0002: Magic numbers

| Property | Value |
|----------|-------|
| **ID** | NUM0002 |
| **Category** | Numeric |
| **Severity** | Warning |
| **Has Code Fix** | Yes |

## Description

Numeric literals (magic numbers) make code harder to understand and maintain. Extract them to named constants with descriptive names.

## Non-compliant Code

```csharp
int timeout = 7777; // NUM0002 - What does 7777 mean?

if (retryCount > 42) // NUM0002 - Why 42?
{
    // ...
}
```

## Compliant Code

```csharp
private const int TimeoutMilliseconds = 7777;
private const int MaxRetryCount = 42;

int timeout = TimeoutMilliseconds;

if (retryCount > MaxRetryCount)
{
    // ...
}
```

## Allowed Values

**Integers:** Small integers (-10 to 20), common values (24, 25, 30, 40, 50, 75), angle values (45, 60, 90, 180, 360), powers of 2 (32, 64, 128, 256, 512, 1024), byte boundaries (255, 65535), time-related (1000, 3600, 86400), hash primes (23, 31, 37, 397).

**Floats:** Zero/one (0.0, 1.0), common fractions (0.01, 0.1, 0.25, 0.5, 0.75), percentage thresholds (0.9, 0.95, 0.99), scientific tolerances (1e-10 through 1e-3).

## Exemptions

- `const` declarations — named constants are the solution
- `static readonly` fields
- Enum values
- Attribute arguments
- Array and collection initializers
- Test methods
- `GetHashCode` methods
- Hex literals (0x prefix)

## When to Suppress

```csharp
#pragma warning disable NUM0002
int specialValue = 12345; // Has a specific meaning documented elsewhere
#pragma warning restore NUM0002
```

## EditorConfig

```ini
dotnet_diagnostic.NUM0002.severity = warning
```
