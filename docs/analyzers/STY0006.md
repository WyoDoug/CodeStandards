# STY0006: Method naming convention

| Property | Value |
|----------|-------|
| **ID** | STY0006 |
| **Category** | Style |
| **Severity** | Warning |
| **Has Code Fix** | Yes |

## Description

Methods must use PascalCase naming.

## Non-compliant Code

```csharp
public void processData() // STY0006
{
}

public int calculate_total() // STY0006
{
}
```

## Compliant Code

```csharp
public void ProcessData()
{
}

public int CalculateTotal()
{
}
```

## Rationale

- Consistent naming across all codebases
- Matches .NET framework conventions
- Improves readability and discoverability

## When to Suppress

Suppress for P/Invoke declarations or interop methods that must match external naming.

```csharp
#pragma warning disable STY0006
#pragma warning restore STY0006
```

## EditorConfig

```ini
dotnet_diagnostic.STY0006.severity = warning
```
