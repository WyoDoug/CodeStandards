# STR0006: No else if

| Property | Value |
|----------|-------|
| **ID** | STR0006 |
| **Category** | Structure |
| **Severity** | Warning |
| **Has Code Fix** | Yes |

## Description

Avoid using else-if chains. Use switch expressions or pattern matching instead.

## Non-compliant Code

```csharp
if (x == 1)
{
    result = "One";
}
else if (x == 2) // STR0006
{
    result = "Two";
}
else
{
    result = "Other";
}
```

## Compliant Code

```csharp
var result = x switch
{
    1 => "One",
    2 => "Two",
    _ => "Other"
};
```

## Rationale

- Switch expressions are more concise
- Compiler exhaustiveness checking
- Cleaner code structure

## When to Suppress

Suppress when conditions are complex boolean expressions that cannot be expressed as switch patterns.

```csharp
#pragma warning disable STR0006
#pragma warning restore STR0006
```

## EditorConfig

```ini
dotnet_diagnostic.STR0006.severity = warning
```
